# Contributing to personalized-education-matrix

Thank you for contributing to the **Omniscient Civilization Nexus (OCN)**.

## Reporting Issues

Search [existing issues](https://github.com/GALACTIC-UNION/personalized-education-matrix/issues) first, then open one using the appropriate template.

## Pull Requests

1. Fork and branch from `main`: `git checkout -b feature/your-feature`
2. Write tests in `tests/`.
3. Pass tests: `pytest tests/ -v`
4. Lint: `flake8 src/ --max-line-length=120 && black src/ tests/`
5. Commit with [Conventional Commits](https://www.conventionalcommits.org/) and open a PR against `main`.

## Commit Types

`feat` · `fix` · `docs` · `test` · `refactor` · `chore`

## Review

Minimum one core maintainer approval. All CI checks must pass.
